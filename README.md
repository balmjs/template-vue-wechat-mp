# BalmJS template - vue-wechat-mp

> A Vue.js project for WeChat Mini Program

## Installation

```sh
npm install -g balm-cli
balm init vue-wechat-mp my-project

cd my-project
yarn
# or
npm i
```

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```

- 使用小程序开发者工具导入项目 `dist/mp`，之后便可预览构建好的小程序了。
